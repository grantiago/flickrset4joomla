This is the install package for a joomla 3.x.
Here you will find the component and plugins of Flickrset4Joomla.

***** PREREQUISITS *****
- Joomla 3.x
- mysql >= 5.0.0
- PHP >= 5.3.1

***** INSTALLATION *****
To install Flickrset4Joomla extract the component and plugins zip-files and install them
with the joomla installer.
If you install each zip file seperatly you must first install the component and afterwards the plugins.
You can also install the whole suite with all extensions trough the joomla installer. After 
a successfull installation configure the 'Add Flickrset button'-plugin and publish him.
